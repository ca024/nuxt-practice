# Nuxt Practice

### Create a Nuxt app

```bash
npx nuxt init <app-name>
```

### Install all dependencies
```bash
npm install
```

### Start development server
```bash
npm run dev
```

### Add Tailwind CSS
```bash
npm install @nuxtjs/tailwindcss
```
Add '@nuxtjs/tailwindcss' to the modules in nuxt.config.ts
